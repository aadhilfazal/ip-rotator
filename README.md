# IP ROTATOR

Automated Tor exit IP rotation tool with live terminal dashboard.

## Features

- Automatic Tor IP rotation
- Live dashboard using Rich
- IP verification
- Rotation history
- Tor ControlPort support

## Requirements

```bash
pip install requests rich stem
```

## Usage

```bash
python ip_rotator.py
```

## Example

```bash
python ip_rotator.py -i 30
```

Rotate IP every 30 seconds.
